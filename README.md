## Run Locally

Clone the project

```bash
  git clone https://github.com/Sasuzin/Desafio-l5-Networks
```

Go to the project directory

```bash
  cd Desafio-l5-Networks
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
